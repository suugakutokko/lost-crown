# Lost Crown（消えた王冠の旅） - ゲーム仕様書

## ゲーム概要

### タイトル
**Lost Crown（消えた王冠の旅）**

### 世界観サマリー
ドラクエ5に近い温かみのある中世ファンタジー世界。
悪の大魔王「マルボルド」が神の王冠を奪い、王国を混乱させた。
プレイヤーは「英雄」として、仲間を集めながら6つのエリア（王国・森・廃墟・砂漠・氷の山・暗黒の城）を旅し、王冠を取り戻す冒険に出る。

戦闘には「爽快感」を最も重視。派手なエフェクト、テンポの良いバトル、必殺技の演出で楽しさを追求する。

## 各画面の仕様

### 1. タイトル画面
- ゲームタイトル「Lost Crown」を大きく表示
- サブタイトル「消えた王冠の旅」
- 背景：王国の城のイラストまたはグラデーション
- ボタン：「冒険を始める」「コンティニュー（セーブデータがある場合）」
- BGM：壮大なオープニング曲（オプション）

### 2. キャラ選択画面
- 3人のプレイヤーキャラクター候補を横並びで表示
- 各キャラクターの情報：
  - 名前
  - 職業（title）
  - 絵文字アイコン
  - 説明文
  - 基本ステータス（HP, MP, ATK, DEF, SPD）
  - スキル一覧
- 選択ボタン「このキャラで冒険を始める」
- 背景：選択に集中できるシンプルな背景

### 3. マップ画面
- エリア一覧を縦スクロール可能なリストで表示
- 各エリアカード：
  - エリア名
  - 絵文字
  - 説明文
  - ステータス：「探索可能」「ロック中」「クリア済み」
- プレイヤーステータス表示（上部固定）：
  - キャラクター名
  - レベル
  - HP/MP
  - 所持金
  - 経験値バー
- メニューボタン：
  - アイテム（回復アイテムの使用）
  - セーブ
  - 設定
- エリア選択で戦闘画面へ遷移

### 4. 戦闘画面
- 上部：敵の情報
  - 敵の絵文字（大きく）
  - 敵の名前
  - HP/MPバー
- 中央：戦闘ログ（最新3～5件のアクションログ）
- 下部：プレイヤー情報
  - プレイヤーの絵文字
  - プレイヤーの名前
  - HP/MPバー（数値表示あり）
- アクションボタン：
  - 通常攻撃
  - スキル1
  - スキル2
  - 必殺技（ゲージが溜まったら）
  - アイテム
  - 逃げる（ボス戦以外）
- エフェクト：
  - ダメージ数値のポップアップ（大きく・クリティカルは赤色）
  - スキル発動時のフラッシュ効果
  - 必殺技の特別演出

### 5. 勝利画面
- 「勝利！」の大きな文字
- 獲得した報酬：
  - 経験値
  - ゴールド
  - レベルアップ情報（該当する場合）
- 新しいエリアのアンロック通知（該当する場合）
- ボタン：「マップに戻る」

### 6. エンディング画面
- 「おめでとう！」のメッセージ
- エンディングストーリー：
  - 王冠を取り戻し、王国に平和が戻った
  - プレイヤーは英雄として称えられる
- プレイ統計：
  - 最終レベル
  - 倒した敵の数
  - 所持金
- ボタン：「タイトルに戻る」

## 開発優先順位（3段階）

### 【優先度1：コアループの確立】
**対象：** タイトル画面 → キャラ選択 → 戦闘画面（基本） → 勝利画面

**理由：**
- ゲームの核となる「選ぶ → 戦う → 勝つ」のループを最優先で作る
- 戦闘の爽快感を早期に体験可能にし、ゲームの面白さを検証する
- 1体の敵との戦闘が完成すれば、残りは敵データの追加だけで拡張可能

**成果物：**
- 基本的な戦闘システム（攻撃・スキル・ダメージ計算）
- 3キャラクターから1人を選べる
- 1種類の敵との戦闘が完全に動作

---

### 【優先度2：戦闘の完成とコンテンツ拡充】
**対象：** 全敵種類の追加・AI実装・必殺技演出・レベルアップシステム

**理由：**
- 戦闘の多様性と深みを追加する
- 敵AI（ランダム、強攻撃優先、回復優先）で戦略性を生む
- 必殺技の派手な演出で「爽快感」を最大化
- レベルアップによる成長実感を与える

**成果物：**
- 5種類の敵（雑魚3、中間ボス1、最終ボス1）
- 各キャラクターの必殺技が完全に実装
- レベル1～10の成長システム
- 敵AIの実装

---

### 【優先度3：マップとゲーム進行の実装】
**対象：** マップ画面・エリア進行・セーブ/ロード機能・エンディング

**理由：**
- マップは戦闘が完成してから追加する「繋ぎ」の役割
- セーブ機能は最後に追加しても問題ない（LocalStorageで簡単に実装可能）
- エンディングは全コンテンツが揃ってから作る

**成果物：**
- 5エリアのマップ画面
- エリアのアンロック進行システム
- セーブ/ロード機能（LocalStorage）
- エンディング画面

## 技術仕様

### プラットフォーム
- 単一HTMLファイル（game.html）
- スマホブラウザ対応（375x812px）
- 技術スタック：HTML + CSS + JavaScript（Vanilla）

### データ形式
- 全ゲームデータは game_data.json に格納
- JavaScript側で `JSON.parse()` で読み込み

### セーブデータ
- LocalStorageに保存
- 保存内容：
  - 選択キャラクターID
  - 現在のレベル
  - 現在のHP/MP
  - 所持金
  - クリア済みエリアID
  - 所持アイテム

## 開発後の期待体験

プレイヤーは自分の好きなキャラクターを選び、個性豊かな敵との戦闘を楽しむ。
スキルと必殺技を駆使して、派手なエフェクトと共に敵を倒す爽快感を味わう。
エリアを進めながらレベルアップし、最終的に大魔王を倒して王冠を取り戻す達成感を得る。

シンプルだが完成度の高い、スマホで手軽に遊べるRPGゲーム。
